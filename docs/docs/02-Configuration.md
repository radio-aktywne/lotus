## Environment variables

You can configure the app at runtime using various environment variables:

- `LOTUS__APIS__ICANHAZDADJOKE__HOST` -
  host of the icanhazdadjoke API
  (default: `icanhazdadjoke.com`)
- `LOTUS__APIS__ICANHAZDADJOKE__PATH` -
  path of the icanhazdadjoke API
  (default: ``)
- `LOTUS__APIS__ICANHAZDADJOKE__PORT` -
  port of the icanhazdadjoke API
  (default: ``)
- `LOTUS__APIS__ICANHAZDADJOKE__SCHEME` -
  scheme of the icanhazdadjoke API
  (default: `https`)
- `LOTUS__APIS__PELICAN__HOST` -
  host of the pelican API
  (default: `localhost`)
- `LOTUS__APIS__PELICAN__PATH` -
  path of the pelican API
  (default: ``)
- `LOTUS__APIS__PELICAN__PORT` -
  port of the pelican API
  (default: `10200`)
- `LOTUS__APIS__PELICAN__SCHEME` -
  scheme of the pelican API
  (default: `http`)
- `LOTUS__DEBUG` -
  enable debug mode
  (default: `true`)
- `LOTUS__SERVER__HOST` -
  host to run the server on
  (default: `0.0.0.0`)
- `LOTUS__SERVER__PORT` -
  port to run the server on
  (default: `10230`)
